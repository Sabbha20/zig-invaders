# zig-invaders

A simple space invaders game written in Zig
## Overview
This is the learning project for Zig Language.
Refering repository:
[bradcypert/zig-invaders](https://github.com/bradcypert/zig-invaders)

### started
```
mkdir zig-invaders
cd zig-invaders
zig init
```
### dependencies
 for graphics it uses raylib-zig bindings for raylib
 ```
 zig fetch --save git+https://github.com/raylib-zig/raylib-zig#devel  
 ```
